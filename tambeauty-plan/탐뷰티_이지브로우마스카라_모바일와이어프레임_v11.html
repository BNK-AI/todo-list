SEC 03 — MOBILE DELIVERY CHECKLIST

모바일 납품 체크리스트

  • 뷰포트 360px 기준 모든 요소 깨지지 않음 · 375/390px에서도 검증
  • 가로 스크롤 금지 — overflow-x: hidden 확인
  • 터치 타겟 최소 44px · 권장 48px · CTA 56px · 인접 8px 간격
  • 플로팅 CTA Cafe24 스킨 기본 처리 — 상세 HTML에서 별도 구현 금지
  • 이미지 최적화 WebP 품질 75% · 최대 400px 폭 · lazy-loading · priority hint for LCP
  • GIF → WebM 우선 (크기 30~50% 절감) · fallback으로 GIF 병행
  • 폰트 로딩 preconnect Google Fonts · font-display: swap
  • 첫 스크린 LCP 2.5초 이내 · 가격/CTA 모두 위쪽 (above fold)
  • CLS(레이아웃 시프트) 0.1 이하 · 이미지 width/height 고정
  • 가로모드 (랜드스케이프) 지원은 선택 · 세로 고정도 가능 명시
  • 접근성 alt 텍스트 모든 이미지 · 명암 대비 WCAG AA (4.5:1)
  • 스크린리더 CTA 버튼 aria-label · 가격/할인 읽기 가능
SEC 04 — BREAKPOINTS & GRID

반응형 브레이크포인트

  • 360~479px (모바일 S) — 1단 · 컬러카드 1~2열 · 폰트 스케일 -1
  • 480~767px (모바일 L / 소형 태블릿) — 1단 · 컬러카드 2열 · 기본 스케일
  • 768~1023px (태블릿) — 1단 유지 · 컬러카드 3열 · 여백 확대
  • 1024px 이상 (데스크톱) — max-width 1040~1240px 중앙 · 3컬럼 섹션카드(v10 핸드오프 준수)
  • 그리드 4열 → 8열 → 12열 순 / 거터 16→24→32px
  • 여백 화면 가장자리 최소 16px (safe) · 24px 권장 (breathing)
