체급 재설정: 잉가 레벨이 정답

이전 벤치마크는 너무 멀었다

데이지크(136K), 힌스(121K), 페리페라(43K), 롬앤(351K)은 탐뷰티 현재 체급과 2~3년 차이. 이들을 따라하면 자원 낭비 + 속도 불일치가 생깁니다.

대신 잉가(@inga_jp 3,412명) 수준에서 움직이는 K-뷰티 5개를 찾았습니다. 모두 인스타 2K~17K, 큐텐 의존형, 히어로 1개 집중으로 움직이는 진짜 동급 브랜드.

이 페이지는 이들의 실제 플레이를 기반으로 탐뷰티가 6개월 안에 실행 가능한 마케팅 플로우를 구체화합니다.

체급 필터 기준

  • 일본 인스타 팔로워 20K 이하
  • 큐텐 메인 채널 (오프라인 부수적)
  • 히어로 SKU 1~3개 집중
  • 2022~2024 일본 진입

제외된 대형 브랜드

  • FWEE (160K) - 이미 졸업
  • AMUSE (52K) - 너무 큼
  • 밀크터치 (45K) - 다음 단계
  • 롬앤/데이지크 - 3~5년 목표

기준선 (잉가)

  • @inga_jp 3,412명
  • LIPS 859건 / @cosme 114건
  • 히어로: 워터 그로우 립틴트
  • 2022년 일본 진입

잉가급 브랜드 공통 플레이 패턴

  • 인스타 팔로워 절대수는 작다 (2K~17K) → 바이럴 콘텐츠 1~2개가 체급보다 중요
  • 히어로 SKU 1~2개가 전체 매출의 70%+ → 라인업 확장보다 히어로 강화
  • 큐텐이 주 전장 → 자사몰/아마존은 후순위. 메가와리 4회가 연간 핵심 이벤트
  • 오프라인은 "상징적 1~5개 매장" → 전국 확장 아님. 플래그십 존재 자체가 브랜드 증명
  • "한 단어"로 설명되는 차별점 (퍼스널컬러/비건/스틱/디저트/글리터) → 탐뷰티도 1단어 필요
  • 리뷰 플랫폼에 등록만 해도 경쟁 우위 → 대형 브랜드 대비 리뷰 숫자가 낮음

탐뷰티 마케팅 플로우 (잉가급 현실 버전)

전략 원칙

대형 브랜드 플레이북을 따라가지 않습니다.
잉가·MERZY·언리시아 수준의 실제 플레이만 참고해, 6개월 안에 실행 가능한 단계로 구성합니다.
목표는 12개월 후 잉가 수준(인스타 3K, LIPS 800+, 큐텐 카테고리 1위) 도달.

1

PHASE 1: 기반 세팅 (1~2주 / 즉시 실행)

잉가급 플레이의 전제조건. 이 단계를 건너뛰면 이후 시딩/프로모가 전환되지 않습니다.

잉가 근거: @inga_jp / @cosme 공식 등록 / LIPS 제품 등록 모두 사전 완료 후 시딩 시작.
실행 액션
  • @tambeauty_jp 일본 전용 인스타 계정 개설 (바이오 일본어, 프로필 링크 큐텐)
  • @cosme에 Cloud Blur 립틴트 제품 등록 (제품 페이지 생성이 리뷰의 전제)
  • LIPS 제품 등록 + 브랜드 페이지 생성
  • 큐텐 공식숍 매장 페이지 정비 (일본어 상세 설명, 리뷰 캠페인 섹션)
2

PHASE 2: 히어로 1개 확정 + 한 단어 포지셔닝 (2~4주)

리소스를 Cloud Blur 립틴트 하나에 집중. 컬러는 이후 확장하되, 지금은 메가데뷔 4위 달성 컬러부터.

근거: MERZY=Velvet Tint, 잉가=Water Glow, 언리시아=Glitter. 모든 잉가급 브랜드가 제품 1개 + 키워드 1개로 움직임.
실행 액션
  • Cloud Blur 립틴트 1개 제품을 "일본 공식 히어로"로 지정 (마케팅 전부 이 제품에 몰빵)
  • 일본어 카피 "クラウドブラー" 키워드 등록 및 모든 콘텐츠에 반복 노출
  • 컬러 TOP 3 선정 (메가데뷔 판매량 상위 + 일본인 선호 예측)
  • 크리에이티브 가이드 1페이지 제작 (스와치·텍스처 표현 규칙)
3

PHASE 3: 마이크로 시딩 30명 (1~2개월)

잉가·언리시아 모두 팔로워 3천~1만 수준. 대형 인플루언서 불필요. 마이크로 인플루언서 30명 릴스 중심 시딩이면 충분.

근거: 언리시아 일본 인스타가 2,467명인데도 큐텐 카테고리 존재감 확보. 소규모 브랜드는 팔로워 체급이 아니라 콘텐츠 품질로 승부.
실행 액션
  • 마이크로 인플루언서 30명 리스트업 (기준: 평균 릴스 조회수 1,000+, 뷰티 관련 콘텐츠 70% 이상)
  • 제품 증정 + 릴스 1개 의무 조건 (해시태그 #PR #タムビューティー #クラウドブラー)
  • @cosme 모니터 캠페인 동시 진행 (30명 체험단 → 리뷰 의무)
  • LIPS PR 리뷰 시딩 (동일 인플루언서 풀 재활용)
4

PHASE 4: 6월 메가와리 집중 공략 (2~3개월차)

잉가의 신제품 프로모 당일 1위 달성 케이스처럼, 시딩 완료 시점(5월말)에 누적된 리뷰/콘텐츠를 메가와리로 전환.

근거: 잉가 워터 더블래스팅 센티드 틴트 = 프로모 시작일 큐텐 포인트메이크 1위. 기존 시딩 → 메가와리 타이밍에 판매량 폭발.
실행 액션
  • 메가와리 1개월 전(5월)에 쿠폰+번들 구성 확정 (20% 쿠폰 + 2개 구매시 미니 증정)
  • 메가와리 시작일 기준 -7일부터 인플루언서 릴스 연쇄 업로드
  • 큐텐 공식숍 베너/상세페이지 메가와리 버전으로 전환
  • 목표: 카테고리 TOP 20 진입 (잉가도 초기 이 단계에서 상승)
5

PHASE 5: 리뷰 100건 돌파 (4~6개월차)

@cosme 리뷰 100건은 일본 소비자 신뢰의 마지노선. 잉가가 114건으로 바이럴 시작.

근거: 잉가 @cosme 114건에서 "최근 은밀하게 엄청 바이럴 되는 코스메" 리뷰 원문 등장. 신뢰도 임계점 = 100건.
실행 액션
  • @cosme 모니터 2차 캠페인 (Phase 3 인원의 2배 규모)
  • 큐텐 구매자 대상 리뷰 작성 쿠폰 제공 (@cosme/LIPS 인증 필수)
  • 인플루언서 2차 계약 (1차 인원 중 성과 상위 10명만 선별)
  • 목표: @cosme 50~100건, LIPS 300~500건
6

PHASE 6: 오프라인 1~3개 매장 테스트 (6~9개월차)

잉가급에서 오프라인은 전국 확장 아니라 상징적 플래그십. 로프트 1~3개 매장 테스트 입점이 현실 목표.

근거: 릴리바이레드(인스타 9.5K)도 로프트 일부 입점에 그침. 잉가도 37개 매장이 2024년부터 시작된 확장.
실행 액션
  • 로프트 시부야/긴자/이케부쿠로 3개 매장 테스트 입점 제안서 준비
  • 큐텐 랭킹/리뷰 숫자 데이터 첨부 (오프라인 바이어 설득용)
  • 입점 시 매장 단위 샘플링 이벤트 (매장별 Cloud Blur 체험존)
  • 목표: 1개 매장이라도 확보 → 9월 메가와리 온오프 연계
